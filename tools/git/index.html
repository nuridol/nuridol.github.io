ScummVM-kor

1.9.0(2016-10-17)을 기반으로 ScummVM kor.에 있는 한국어 출력 기능을 추가해서 수정/변환한 버전입니다.
소스코드 https://github.com/nuridol/scummvm-kor

로지텍 K810, K811용 FN키 설정 프로그램

로지텍 프로그램을 설치하지 않고도 FN키 없이 펑션키를 쓸 수 있게 키보드에 코드를 적용하는 프로그램입니다.

  • macOS용
    K810용 K810FnKey.app
    K811용 K811FnKey.app
  • Windows용
    K810용 K810FnKey.exe
    K811용 K811FnKey.exe
